Android vs iOS

Intro

Although both operating systems are very similiar in the sense that they mostly provide the same functions - making calls, sending messages, taking photos and videos, there is a lot of debate around which is better. The choice is in most cases very subjective, based on the design and maybe popularity of some phone with the regarded operating system.

Basic differences

One plus for android based phones is that there are so many to choose from, while there are not so many options with the iOS system (mostly phones with a different following number - iPhone X, iPhone 5, ...). Android phones are much more flexible, you can change the look of the phone home screen in many ways, apply various themes. That can be good or bad since it can slow the phone down significantly, which doesn't happen with iPhones, as they don't have so many customizing features.

Apps

Although popular apps are mostly available on both platforms, in the recent time new apps generally first appear on the iOS store. Many google apps are available on the iOS as well, while in contrast only the Apple music app is available on the android system.

Conclusion

There is no real telling which phones are better, it is mostly a personal choice. There are good phone on both ends, so you can't really go wrong here. Personally, I like android based phones, just because I believe that iOS phones are very overpriced along with all their silly additional accessories, while on the other hand you can get the same build with an android phone with a far smaller budget.

Why has machine learning become so popular?

Intro

Machine learning has been surely the topic of the century in the IT sector. Although it has been deeply researched even in the 90s, is hasn't been until today that machine learning has shown it's true power. That is mostly due to the developement of powerful computers and the rich data sources required to build models that solve problems in high-dimensional space.

History

To understand the present, we have to learn about the past. Machine learning is, in part, based on a model of brain cell interaction, which gave name to the term "neural network", very common in the topic of machine learning. That model was created by Donald Hebb ("The Organization of Behavior), which relies on the communication and excitement of neurons. There are many important projects that led to the developement of AI, for example:

  1. The Game of Checkers machine
  2. The Perceptron
  3. The nearest neighboor algorithm

and many more. The discovery of using multilayer perceptron in the 1960s was a big step forward.

Today

Today machine learning can be seen everywhere, in sales forecasting, speech recognition, facial recognition, fraud detection, natural language processing... With the developement of computers in general, machine learning will only become more and more powerful. Modern machine learning models can be used to predict events ranging from disease outbreaks to the rise and fall of stocks.

Conclusion

Recently, Machine Learning was defined by Stanford University as “the science of getting computers to act without being explicitly programmed.” It is a very promising field for new IT student to delve into, keeping in mind that it is no easy task, if they plan on doing the big stuff. Surely machine learning will affect our lives in ways we can't imagine yet, and personally, I can't wait for that to happen.
